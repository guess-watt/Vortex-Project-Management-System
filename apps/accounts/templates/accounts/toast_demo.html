How It Works
Automatic Django Messages:

All Django messages (from views using messages.success(), messages.error(), etc.) will automatically appear as toast notifications when the page loads.

Manual JavaScript Usage:
// Basic usage
window.ToastManager.success('Task saved!');
window.ToastManager.error('Failed to delete');
window.ToastManager.warning('Please review');
window.ToastManager.info('Update available');

// With custom duration (milliseconds)
window.ToastManager.show('Message', 'success', 5000);

// With action button
window.ToastManager.show('File uploaded', 'success', 4000, {
    label: 'View',
    onClick: () => window.location.href = '/files/'
});
Features
  • ✅ 4 toast types: success (green), error (red), warning (yellow), info (blue)
  • ✅ Icons: ✅ ❌ ⚠️ ℹ️
  • ✅ Slide-in animation from top-right
  • ✅ Progress bar countdown
  • ✅ Auto-dismiss after configurable duration
  • ✅ Stack multiple toasts with 8px gap
  • ✅ Mobile responsive (full width on screens < 576px)
  • ✅ Optional action buttons
  • ✅ Close button on each toast
  • ✅ Dark mode support
